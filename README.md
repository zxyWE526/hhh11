# 陕西朱鹮酒业有限公司电子商务网站

## 项目概述

这是一个为陕西朱鹮酒业有限公司打造的现代化电子商务网站，展示和销售有机黄酒、黑米酒等产品。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite 
- **UI 组件库**: shadcn/ui (基于 Tailwind CSS)
- **路由**: React Router v6
- **样式**: Tailwind CSS v4

## 项目结构

## 功能特性

### 前台功能

1. **首页** (`/`)
   - 企业 Logo 展示
   - 自动轮播图（3 张）
   - 精选商品展示（6 款）
   - 企业优势展示
   - 合作伙伴链接

2. **商品列表页** (`/products`)
   - 商品分类筛选（7 个分类）
   - 关键词搜索
   - 商品卡片展示（图片、名称、价格、评分）
   - 快速查看详情

3. **商品详情页** (`/product/:id`)
   - 商品大图展示
   - 详细信息（名称、价格、规格、描述）
   - 产品参数（容量、酒精度、保质期等）
   - 评价星级
   - 数量选择
   - **加入购物车功能**
   - 相关产品推荐

4. **购物车** (`/cart`)
   - 查看购物车商品
   - 增减商品数量
   - 删除商品
   - 订单摘要（总额计算）
   - 立即结算

5. **订单结算** (`/checkout`)
   - 确认订单信息
   - 查看商品清单
   - 提交订单（模拟）
   - 订单成功页面

6. **用户系统**
   - **注册页** (`/register`): 表单验证（用户名/邮箱/密码/手机）
   - **登录页** (`/login`): 表单验证、记住我、社交登录入口
   - **我的订单** (`/orders`): 查看用户订单历史
   - localStorage 模拟登录状态
   - 导航栏显示用户信息

7. **新闻资讯** (`/news`)
   - 新闻列表（6 条新闻）
   - 新闻分类标签
   - 阅读次数显示
   - **新闻详情页** (`/news/:id`): 完整新闻内容

8. **关于我们** (`/about`)
   - 企业故事
   - 企业使命（有机认证、传统工艺、生态种植）
   - 酿造工艺流程
   - 联系方式

9. **联系我们** (`/contact`)
   - 公司信息（地址、电话、邮箱、营业时间）
   - **在线留言表单**（姓名、电话、邮箱、公司、留言内容）
   - 留言提交成功提示

### 后台管理

1. **后台登录** (`/admin/login`)
   - 管理员账号密码登录
   - 默认账号：`admin` / `admin123`
   - 权限验证提示

2. **管理后台** (`/admin/dashboard`)
   - **总览**: 销售额、订单数、待处理订单、留言数统计
   - **订单管理**: 查看所有订单、订单详情、订单状态
   - **用户留言**: 查看用户在线留言
   - **商品管理**: 查看商品列表（6 款商品）

### 设计特色

- **Logo 设计**: SVG 绘制朱鹮 + 禾苗融合图案，绿色 + 金色配色
- **主题色**: 清新绿色系 (#2D5F40 ~ #4A8C60) + 金色 (#D4AF37)
- **动画**: 轮播图、卡片悬停浮动、页面渐入、有机曲线变形
- **响应式**: 支持桌面端和移动端
